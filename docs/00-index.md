# Uni-app 切片索引

## 使用方式

- 先按任务主题定位文档。
- 每次只加载必要切片，减少上下文噪音。
- 执行前提炼“必须规则”，执行后对照“检查项”。

## 切片目录

- `01-项目结构.md`：目录分层、模块边界、命名约定
- `02-路由与页面.md`：`pages.json`、页面组织、导航规范
- `03-状态与缓存.md`：Pinia、storage、登录态缓存策略
- `04-请求与拦截器.md`：请求封装、鉴权头、统一错误处理
- `05-组件与样式规范.md`：组件拆分、样式命名、层级控制
- `06-多端差异处理.md`：条件编译、平台能力差异与降级
- `07-打包发布.md`：环境变量、构建命令、发布检查
- `08-常见问题.md`：高频故障与排查顺序
- `09-App原生插件与能力.md`：Android/iOS 原生插件、基座调试、鸿蒙路线边界
- `10-页面样式与布局.md`：CSS 语法边界、单位规则、跨端样式兼容与排障
- `11-Vue基础语法.md`：模板指令、响应式数据、事件与列表渲染规范
- `12-Vue组件通信与复用.md`：props/emits/slot 语义边界与组件复用策略

## 业务切片目录

- `business/00-index.md`：业务切片导航与填写顺序
- `business/01-业务模型模板.md`：实体、状态、字段、业务规则模板
- `business/02-接口契约模板.md`：接口入参、出参、错误码、权限模板
- `business/03-业务流程模板.md`：页面流程、分支、前后置条件模板
- `business/04-业务验收模板.md`：验收用例、边界场景、回归清单模板

## 维护约束

- 新规则必须归档到单一切片，禁止跨文件复制。
- 切片结构固定：适用范围 -> 规则 -> 示例 -> 反例 -> 检查项。
- 发生规则变更时，同步更新时间与影响范围说明。
