# SliceLab AGENTS 规则

## 适用范围

- 本文件规则仅对 `SliceLab` 项目生效。

## 基础约束

1. 所有实现必须符合 `docs/` 规范。
2. 新增业务能力必须同步更新 `docs/business/` 对应文档。
3. 禁止跨目录随意新增文件，优先复用既有模块结构。
4. 页面不允许直接堆复杂 Canvas 逻辑，核心算法必须模块化封装。
5. 导入路径优先使用 `@/`，避免深层相对路径。

## Slice 模块约束

1. 切片核心逻辑统一封装在 `src/utils/image.ts`（若文件不存在则创建）。
2. 页面层只做参数收集、流程编排和结果展示，不直接写复杂像素处理。
3. 任何影响切片结果的规则变更，必须更新 `docs/business/10-slice-module.md`。

## 交付约束

1. 改动前先说明思路并等待确认。
2. 改动后需给出受影响文件和验证结论。
